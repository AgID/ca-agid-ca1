Questo repository contiene il certificato della CA authority AGID CA1, preposta ad emettere i certificati per il circuito PEC; contiene inoltre il certificato della CA AGID CA SSL SERVER preposta a emettere certificati SSL Server per i domini sui quali AGID esercita il controllo. Oltre al certificato della CA sono presenti i file necessari alle verifiche di integrità e autenticità del certificato.
